도메인 이벤트 (EDA)

컨텍스트 간 통신은 이벤트로. 스케줄러(BullMQ/@Interval)가 예약 시각에 방을 전환, 종료 시 점수·포인트·알림이 이벤트로 연쇄.

상태 머신

두 핵심 라이프사이클: 방(Room)과 경주 참가자(RaceParticipant).

Room 상태

RECRUITING ▶예약 T-Δ STARTING ▶카운트다운 LIVE ▶전원완주/시간초과 FINISHED CANCELLED ▶ RECRUITING/STARTING 2명 미만

RaceParticipant 상태 (LIVE 중)

RUNNING ▶목표거리 FINISHED RUNNING ▶시간초과/포기 DNF RUNNING ⇄끊김/복귀 DISCONNECTED
DISCONNECTED 중에도 로컬 GPS는 계속 거리 측정 → 복귀 시 재동기화. DNF도 달린 거리는 점수에 반영(완주 보너스만 0).
