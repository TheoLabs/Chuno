추노 백오피스 MVP 로드맵

사용자 트랙과 평행한 관리자단 로드맵. IAM/role-RBAC 기반 위에 사용자 관리·모더레이션·게임 운영·포인트/랭킹/공지를 쌓는다.

🎫 상세 이슈 보드 (BO 스텝별 티켓) →

기술스택 노트 — back-office = React + TypeScript + React Query + React Router DOM + Ant Design(별도 웹 SPA). 모든 관리자 API는 /api/admins/* + AdminGuard, 인증 = 외부 SSO, 권한 = role-only RBAC(SUPER_ADMIN / OPERATOR / SUPPORT).
