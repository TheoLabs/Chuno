추노 백오피스 와이어프레임

관리자 전용 React + TypeScript SPA(React Query · React Router · Ant Design)의 정적 화면 목업. 사용자 앱의 다크+코랄 "추격전" 정체성과는 완전히 다른 표면 — 라이트 테마의 표준 어드민 콘솔(Sider·Header·Table·Form·Modal)이다. 좌측 메뉴로 9개 화면을 이동할 수 있고, 각 화면 상단의 칩은 관련 백오피스 이슈로 연결된다.
